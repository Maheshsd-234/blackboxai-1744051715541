Technology Stack Breakdown

A robust combination of blockchain and modern web technologies.

Ethereum Blockchain

Smart contracts manage access control and log all transactions immutably.

Key Features:

  • Decentralized access control
  • Tamper-proof audit trail
  • Patient-owned identity

IPFS Storage

Medical files are stored decentralized with content-addressable hashes.

Key Features:

  • No single point of failure
  • Content-addressable files
  • Encrypted end-to-end

MetaMask SDK

Web3 authentication and transaction signing through user wallets.

Key Features:

  • Non-custodial identity
  • One-click authentication
  • Cross-device compatibility

React Frontend

Modern, responsive user interface built with React and Tailwind CSS.

Key Features:

  • Component-based architecture
  • Mobile-first responsive design
  • Optimized performance

Node.js Backend

API services and IPFS pinning nodes for reliable file storage.

Key Features:

  • High-performance API
  • IPFS cluster management
  • Event-driven architecture

Zero-Knowledge Proofs

Advanced cryptography for privacy-preserving verification.

Key Features:

  • Verify without revealing data
  • Age/condition verification
  • Prescription validation
